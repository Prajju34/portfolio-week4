# MyBiz Blog — Portfolio Project

## Overview
Responsive blog-style business website built with HTML, CSS and JavaScript.
Pages: Home, About, Blog, Contact.

## Files
- index.html
- about.html
- blog.html
- contact.html
- style.css
- main.js
- images/ (put hero.jpg, post1.jpg, post2.jpg, post3.jpg)

## Run locally
1. Open folder in VS Code.
2. Right-click `index.html` → Open with Live Server.
3. Visit http://127.0.0.1:5500/

## Notes
- Replace mailto address in contact.html to receive emails.
- Ensure your images are in images/ with exact filenames (lowercase).
