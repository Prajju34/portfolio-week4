# MyBiz Blog — Portfolio Project

## Overview
Small responsive blog-style business website built with HTML, CSS and JavaScript.
Pages: Home, About, Blog (3 posts), Contact (form validation).

## Run locally
1. Open this folder in VS Code.
2. Right-click `index.html` → Open with Live Server.
3. Visit `http://127.0.0.1:5500/`.

## Files
- index.html, about.html, blog.html, contact.html
- style.css
- main.js
- images/ (hero.jpg, post1.jpg, post2.jpg, post3.jpg)
